# Bank Term Deposit Prediction using Machine Learning

## Project Overview

This project predicts whether a customer will subscribe to a bank term deposit using Machine Learning techniques. The project includes data preprocessing, exploratory data analysis, feature engineering, visualization, and predictive modeling using Logistic Regression.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Algorithm

- Logistic Regression

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Categorical Encoding
5. Feature Scaling
6. Train-Test Split
7. Logistic Regression Model Training
8. Model Evaluation
9. Prediction Analysis

---

- Data preprocessing

## Dataset

The dataset contains customer-related information such as:

- Age
- Job
- Marital Status
- Education
- Contact Type
- Campaign Information
- Call Duration
- Previous Marketing Outcomes

Target Variable:
- Subscription to term deposit (`y`)
